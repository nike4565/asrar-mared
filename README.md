# مستودع asrar-mared

## 📝 المقدمة
هذا هو المستودع الرئيسي لمجموعة **المارد الرقمي – أسرار مراد**.  
يمثل نقطة التحكم المركزية التي تُدار منها جميع المشاريع والفروع، وهو القاعدة الأساسية التي تجمع المساهمين تحت راية واحدة.  
كل التوجهات التقنية، القرارات الاستراتيجية، والميول المستقبلية تبدأ من هنا.

---

## 🎯 الهدف
- توفير بيئة منظمة وموحدة لإدارة المشاريع.  
- تمكين المساهمين من العمل تحت فرع رئيسي واضح (**Warrior**).  
- ضمان استقرار الكود وتكامل خطوط النشر (CI/CD).  
- بناء مجتمع قوي يشارك في تطوير وإدارة المشروع بشكل مفتوح وشفاف.  

---

## ⚙️ البنية
- **الفرع الرئيسي:** `Warrior`  
- **الأرشيف:** `main` (للتاريخ فقط)  
- **CI/CD:** جميع الـ Workflows محدثة للعمل على الفرع الجديد.  

---

## 🚀 كيفية المساهمة
1. تأكد أنك تعمل على الفرع **Warrior**.  
2. قم بعمل Fork أو Clone للمستودع.  
3. أضف مساهمتك عبر فرع فرعي (Feature Branch).  
4. افتح طلب سحب (Pull Request) مع وصف واضح للتغييرات.
